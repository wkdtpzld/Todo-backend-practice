# Todo-backend-practice

간단한 일정 체크 웹 페이지 입니다. 기능적인 면에서는 별 의미가 없는 토이 프로젝트입니다만.

Basic Auth를 이용한 로그인을 사용하지 않고 보안성이 살짝 더 좋으며
스케일이 늘어났을 경우를 상정하였고

가장 적합한 것이 JWT 토큰이었습니다.

또한 React.Js와 REST통신을 하기떄문에 프론트 링크를 따로 적어놓겟습니다.
